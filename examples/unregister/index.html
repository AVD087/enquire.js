Unregister Example

This example shows how you can use enquire.js' unregister method to remove media queries. You can either remove an entire query and all handlers, or you can remove individual handlers. Here we will remove an entire query.

If you click the button below, the media query will be unregistered, thereby calling it's destroy callback. This particular callback adds the class "destroy" to this paragraph, which is styled to have a strikethrough the text.
