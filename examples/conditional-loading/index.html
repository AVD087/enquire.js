Example of conditionally loading content with enquire.js

This example shows how you can use the setup function to conditionally load content. Whilst this has always been possible with enquire.js, as of v2.0.0 this has been greatly simplified.

The setup callback will conditionally load in a script when the page is greater that 500px in width. This script will add the class "setup" to this paragraph, thereby making it bold. You can confirm this in the network panel in your developer tools.
