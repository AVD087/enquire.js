Example using shouldDegrade flag with enquire.js

This example shows how you can use enquire.js' shouldDegrade flag to deal with incapable legacy browsers. shoudlDegrade will cause any legacy browser that enquire deems to be incapable to have the match callback always run, regardless of whether the media query has actually matched.

This "short circuit" of the media query allows you to use a mobile-first approach whilst still delivering a desktop experience to legacy browsers. For this reason, it's typical that this flag will be set on queries targetting larger viewports. Here the particular media query will only match if the viewport is 500000px wide. It's unlikely you have a viewport that wide (congratulations if you do!!!) so will never see the effects, but if you come here with IE8, for example, you will see that the match callback is executed.
